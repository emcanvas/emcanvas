# 09 — MVP Scope

## Entra en MVP

- Plugin scaffold conceptual con `definePlugin()`.
- Canvas editor con 10 componentes base.
- Drag & drop.
- Selección y property editing.
- Responsive breakpoints: desktop, tablet y mobile.
- Undo/redo.
- Autosave y save manual.
- Preview usando el sistema existente de EmDash.
- Frontend renderer en Astro.
- Entry takeover flow básico.

## Queda diferido

- Template/block library.
- API de custom components third-party.
- Form builder.
- Animations/transitions.
- Collaborative editing.
- Global theme system.
- Import/export de layouts.
- Component marketplace.

## Criterio de recorte

El MVP prioriza cerrar el loop completo de valor: editar, guardar, previsualizar, renderizar y publicar. Todo lo que no aporte directamente a ese loop queda fuera.
