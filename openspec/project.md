# Project: EmCanvas

## Purpose

EmCanvas is a visual page builder for EmDash CMS delivered as an external trusted plugin. It enables drag-and-drop page composition while preserving EmDash as the host CMS and content lifecycle owner.

## Product Direction

- External plugin, not core EmDash.
- Entry-level takeover, not content-type takeover.
- Layout stored in `entry.data`.
- Server-rendered frontend output.
- No upstream modifications to EmDash.

## Current State

The repository is in product-definition phase. This OpenSpec setup captures validated assumptions and prepares future implementation planning.
